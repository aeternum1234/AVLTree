# AVL-Tree
[![Build Status](https://travis-ci.com/ambersun1234/AVL_Tree.svg?branch=master)](https://travis-ci.com/ambersun1234/AVL_Tree)
<br>
simple practice of AVL tree
### Clone repo
```=1
git clone https://github.com/ambersun1234/AVL_Tree.git .
```
### Compile source code
```=1
cd AVL_Tree
make
```
### Running
```=1
./avl_tree
```
### Testing
```=1
make test
```
